# Student OS

Student OS is a platform for MCA students and faculty.

## Tech Stack

* Django
* Django REST Framework
* React
* Vite
* Tailwind CSS
* PostgreSQL
* Google SSO

## Backend Setup

python manage.py migrate
python manage.py runserver

Backend URL:
http://127.0.0.1:8000/

## Frontend Setup

npm install
npm run dev

Frontend URL:
http://localhost:5173/
